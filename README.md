## OFDM system

#### Project Purpose
- To understand the process of OFDM system based on BPSK modulation.

#### Project Procedure
- Generate 8 integers randomly consisting of 0 and 1 
- Create BPSK signal using pskmod function
- Serial to Parallel
- IFFT function
- Parallel to Serial
- PAPR(Peak to Arverage Power Ratio) Check
- Receiver Design
- Serial to Parallel
- FFT function
- Parallel to Serial
- BPSK demodulation
- Check Symbol
